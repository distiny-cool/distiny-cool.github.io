### 起因

最近主要用Ubuntu电脑，手机有时候需要投屏到电脑上，Windows解决方案巨多，但是Ubuntu上面方法有限，记录一下。

### 过程

看了半天，其实很简答啦，就安装一个scrcpy就行

```
sudo snap install scrcpy
```

下面介绍以下怎么用

手机连接电脑，开到USB调试模式

终端输入：

```
scrcpy -wS
# w代表连接后关闭手机屏幕
# S表示连接后手机保持常亮
```

然后就OK啦
